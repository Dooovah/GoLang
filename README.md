# GoLang
This is a GoLang practice repository. [WelcomeUser](./WelcomeUser) is a basic starting point. Below are suggested project generated by Claude to work on.


## Beginner
### Number guessing game
The program picks a random number and the user keeps guessing until they get it right. Covers random number generation, loops, conditionals, and user input.<br>
**Status:** 🔴 Not Started

### Simple calculator
Takes two numbers and an operator from the user and returns the result. Covers user input, switch statements, and basic error handling (e.g. division by zero).<br>
**Status:** 🔴 Not Started

### Temperature converter
Converts between Celsius, Fahrenheit, and Kelvin. Covers functions, packages, and working with floating point numbers.<br>
**Status:** 🔴 Not Started


## Intermediate
### To-do list CLI
A command line app where you can add, remove, and list tasks, saved to a text file so they persist between runs. Covers file I/O, structs, slices, and loops.<br>
**Status:** 🔴 Not Started

### Word frequency counter
Reads a text file and counts how many times each word appears, then prints them sorted by frequency. Covers maps, file reading, string manipulation, and sorting.<br>
**Status:** 🔴 Not Started


## More Challenging
### REST API
A simple HTTP server with a few endpoints (e.g. a basic note-taking API). Covers Go's built-in net/http package, JSON encoding/decoding, and routing.<br>
**Status:** 🔴 Not Started

### Web scraper
Fetches a webpage and extracts specific information from it. Covers HTTP requests, string parsing, and potentially introduces you to third party packages.<br>
**Status:** 🔴 Not Started
